# Free Mock / Test Data API

Free fake/test data API - realistic people, companies, addresses, UUIDs and more as JSON. No key.

**Live tool & docs:** https://mock.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://mock.wrapper-agency.com/api/v1/mock?type=person&count=10&locale=en"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://mock.wrapper-agency.com
